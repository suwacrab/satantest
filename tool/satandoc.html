VDP2 NOTES

ÅñMEMORY MAP

VDP2 VRAM begins at 05E00000H.

Addresses are relative to the beginning of VDP2 VRAM.

VDP2 Description Address Size
VDP2 VRAM Bank A0 00000000H 128 KB
VDP2 VRAM Bank A1 00020000H 128 KB
VDP2 VRAM Bank B0 00040000H 128 KB
VDP2 VRAM Bank B1 00060000H 128 KB
VDP2 Color RAM 00100000H 4 KB
VDP2 I/O register address 00180000H

the four VRAM banks hold data for your character data/tile entries.

Color RAM (CRAM) holds the VDP2's palette, which can be used for both VDP1 and VDP2 screens.

the IO registers configure the VDP2, they're used for changing the screen size, priority, etc.

ÅñI/O REGISTERS

(R) means a register is read-only, (R/A) means reading is allowed, (W) is write-only, and (R/W) means it can be read and written to.

All VDP2 registers are halfwords (16 bits).

00180000 : (R/A) : TV Mode

Bit(s) Name Description
15 DISP : TV screen display bit Controls if picture's displayed on TV screen. (0=no display,1=display)
8 BDCLMD : Border color mode bit Controls the border area's colors. (0=display black,1=display back screen)
7~6 LSMD1,LSMD0 : Interlace mode Designates the interlace mode. (0=non-interlace,2=single-density,3=double-density)
5~4 VRESO1,VRESO0 : Vertical resolution bit Designates vertical resolution of picture displayed on TV screen. (0=224,1=240,2=256(PAL ONLY))
2~0 HRESO2~HRESO0 : Horizontal resolution bit Designates horizontal resolution of picture displayed on TV screen. (0=320,1=352,2=640,3=704,4=320(31kHz),5=352(Hi-Vision),6=640(31kHz),7=704(Hi-Vision))

00180002H : (R/A) : External Signal Enable Register

Bit(s) Name Description
9 EXLTEN : External latch enable bit Selects condition for latching HV counter value to the HV counter register. (0=latch on reading external signal enable register,1=latch through external signal
8 EXSYEN : EXSYNC enable bit Controls input to internal synchronous circuit of external sync signal. (0=does not input external sync signal,1=inputs external sync signal,syncs TV display with external))
1 DASEL : Display Area select bit designates image display area, only valid when EXBGEN is 1. (0=display only in set display area,1=display in standard display area
0 EXBGEN : EXBG enable bit Controls input to external screen data.
